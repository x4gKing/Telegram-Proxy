# 🌐 MTProto Proxy List

لیست پروکسی‌های MTProto تلگرام، به‌صورت **کاملاً خودکار هر ۱۲ ساعت یک‌بار به‌روزرسانی می‌شود**.

بدون تبلیغات، بدون نیاز به ثبت‌نام — فقط یک لیست تازه و تست‌شده از پروکسی‌های عمومی برای اتصال سریع‌تر و پایدارتر به تلگرام.

---

## 📌 چطور استفاده کنم؟

می‌توانید پروکسی‌ها را از اینجا ببینید:

👉 **[https://x4gking.github.io/Telegram-Proxy/](https://x4gking.github.io/Telegram-Proxy/)**

فقط کافی است وارد صفحه شوید، روی یکی از لینک‌های **Open** کلیک کنید — تلگرام به‌صورت خودکار پیشنهاد می‌دهد که آن پروکسی فعال شود.

اگر یک پروکسی کار نکرد، از همان صفحه یکی دیگر را امتحان کنید — چون لیست شامل چندین پروکسی مختلف است.

---

## 🔄 به‌روزرسانی خودکار

این لیست هر **۱۲ ساعت** به‌صورت کاملاً خودکار بازبینی و جایگزین می‌شود، بنابراین همیشه یک نسخه‌ی تازه از پروکسی‌های در دسترس را خواهید داشت.

---

## ⚠️ نکته‌ی امنیتی

پروکسی‌های عمومی توسط اشخاص ثالث ناشناس اداره می‌شوند. محتوای پیام‌های شما همچنان با رمزنگاری end-to-end تلگرام محافظت می‌شود، اما اپراتور پروکسی می‌تواند IP و زمان اتصال شما را ببیند. برای استفاده‌ی حساس‌تر، پروکسی اختصاصی خودتان را ترجیح دهید.
